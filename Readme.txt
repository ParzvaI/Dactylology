# Dactylolgy Reader
***
Web app that uses Tensorflow.JS to detect real time human hand gestures. The detected gesture is then converted into text format and display its exact meaning to the user.
This web app helps the people with difficulty in speaking and hearing to interpret and convey their message in the most convenient manner.

## Technologies
***
A list of technologies used within the project:
* TensorFlow : Version 2.4
* ReactJS    : Version 17.0
* OpenCV     : Version 4.5.2
* IBM Cloud Object Storage : https://www.ibm.com/in-en
